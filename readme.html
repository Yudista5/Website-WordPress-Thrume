# Thru.me - Fashion Brand Company Profile Website ## _Be True, Be You_ ## About Us Thru.me is the company profile website for local fashion brand in Indonesia that supported by WordPress. This brand has purpose want to bring back the traditional fashion to modern women with main tagline "Be True, Be You". The first brand campaign we do with "Cantik dengan Kain" to build awareness and lead generated. ## Features - Landing Page - List of products - Shopping cart - Blog - Payment processing ## How to Use 1. Browse the list of products. 2. Add products to your shopping cart. 3. Checkout and pay for your items. ## Technologies Used - WordPress: Elementor, Woocommerce, Click to chat, ultimate member - CSS for custom elemen ## Requirements * Install WordPress on your web server. * Add few plugins * A database ## Source Code The source code for this project is available on GitHub: https://github.com/Yudista5/Website-WordPress-Thrume
